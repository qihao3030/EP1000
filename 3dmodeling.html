This page is the documentation of the things I learn throughout the course!
3D modeling
For 3d modeling, i will be using fusion 360. As an introduction, we had 2 drawing exercise that are pretty simple as i had previouse experience with fusion 360. These 2 exersice is mainly about setting dimensions, adding constraints and a introduction in simple shapes.
Name Tag
The first excercise is to create a name tag with the given dimensions. Here we learn about how to extrude and create sketch on another plane.
Lego brick
The second exercise is to create a lego brick, to do that, we followed a youtube video made by kevin kennedy. The whole process is simpler than i thought. I simply sketch out the box, added one cylinder then used the rectangular pattern funtion to create all the other cylinders, and repeated the same thing for the top. After that I used the shell funtion to hollow out the lego.
Cube pyramid
The third excersice is to make a cube pyramid. to do this, I first created the first layer, then extude it, then create another layer ontop of that and so on. the pyramid is then hollowed out using the shelled function.
Mug with handle
The fourth excersie is to create a mug with a handle. I first sketched out half of the mug in 2d, then used the revolve tool to create the mug. the same method is used for the handle.
Hex Nut
the fifth exercise is to create a Hex nut. Again we watched a video created by kevin kennedy. this excersise is mainly to introduce the hole function.
The last 3d modeling project is to create something that only can be 3d printed. so i decided to created a ball in cage. The ball is slightly bigger than the box so the ball will not fall out.
To embed my fusion 360 drawing in my webpage
  1. navigate to the model in the Fusion 360 Data Panel and click "View Details on Web".
  2. Select the "Share" icon.
  3. Select "Embed".
  4. Select the size.
  5. Copy the code and paste it into the code of my website.
